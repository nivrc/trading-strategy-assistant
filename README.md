# STRATEX - Trading Strategy Assistant

A full-stack trading strategy backtesting platform that allows users to create, manage, and backtest trading strategies using pre-built templates. Features comprehensive SQL-based reporting and performance analytics.

## üöÄ Features

### Strategy Templates
- **Moving Average Crossover**: Classic trend-following with customizable fast/slow periods
- **RSI Mean Reversion**: Buy oversold, sell overbought conditions
- **MACD Crossover**: Momentum-based entry/exit signals
- **Bollinger Bands**: Mean reversion using volatility bands
- **Dual Momentum**: Combines absolute and relative momentum
- **Mean Reversion**: Z-score based deviation trading

### Backtesting Engine
- Execute strategies on historical price data
- Configurable date ranges and initial capital
- Real-time performance calculation

### Performance Metrics
- Total & Annualized Returns
- Sharpe Ratio (risk-adjusted returns)
- Maximum Drawdown
- Win Rate & Profit Factor
- Trade-by-trade analysis
- Consecutive wins/losses tracking

### Interactive Visualizations
- Equity curve charts
- Drawdown visualization
- Monthly returns bar charts
- Trade history table

### SQL-Powered Reports
Demonstrates advanced SQL skills with complex queries:
- **Strategy Comparison**: Aggregate performance across strategies
- **Performance by Symbol**: Cross-sectional analysis
- **Time Analysis**: Temporal performance patterns
- **Top Performers**: Ranked backtest results
- **Risk Metrics**: Risk-adjusted performance analysis
